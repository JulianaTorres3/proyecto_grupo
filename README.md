Descripcion proyecto: Laboratorio 1 se realizaron los pasos iniciales para el uso de git modificando un archivo index.html cada colaborador desde una rama local usando pull request 

Lista de comandos utilizados:
git add . : Agregar cambios realizados
git commit -m "": Nombrar el cambio
git pull origin : Halar cambios desde una rama 
git push origin : Enviar cambios a una rama
git merge : Fusionar dos ramas sin importar orden
git branch -a : Listar todas las ramas
git branch -d : Eliminar una rama 
git checkout -b : Crear una rama
git checkout : Moverse a una rama
git status :Verificar el estado de los archivos
git rebase : Sobreponer en order consecutivo los cambios de una rama a otra 
git rebase --continue : Aceptar los conflictos para seguir con el rebase 
notepad INDEX.html : Modificar archivo desde block de notas
